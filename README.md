# WifiWala Admin Panel

Modern, responsive admin dashboard built with React + Chakra UI.

## Setup

1. Install Node.js 18+
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```

## Features

- Chakra UI themed with blue `brand` color palette
- Responsive layout with collapsible sidebar + top navbar
- Dynamic routing with React Router
- CRUD modules with reusable table and modal form components
- Form validation, loading states, empty states, and toast feedback
- Optional enhancements included:
  - Dark mode toggle
  - Search + filtering in tables
  - Role-based UI preview in Settings

## Folder Structure

- `src/components` reusable UI and layout
- `src/pages` page modules
- `src/hooks` custom hooks
- `src/utils` helpers/validators
- `src/data` mock data
